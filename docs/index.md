# Getting started

![](img/seito.png)

Simply install `seito` coming directly from pypi with your favorite python package manager. 
This will install all needed dependencies.

## with package manager

=== "poetry"
    ```bash
    poetry add seito
    ```

=== "pip"
    ```bash
    pip install seito
    ```
